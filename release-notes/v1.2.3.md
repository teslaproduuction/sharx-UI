## Release v1.2.3

**Release Date:** April 28, 2026

Патч-релиз относительно **v1.2.2**: исправлено отображение `flow` в подписках для `xhttp` и усилена надежность загрузки geodata во время сборки Docker-образа.

### Subscription и ссылки

- **VLESS/Trojan links:** `flow` теперь добавляется не только для `tcp`, но и для `xhttp` в URL-подписках.
- **JSON subscription (VLESS):** добавлено аналогичное условие по сети (`tcp`/`xhttp`) при добавлении `flow`.
- **Благодарность за патч:** проблема и рабочий фикс подтверждены в issue [#71](https://github.com/konstpic/SharX/issues/71), автор предложенного патча — **@silverfox0000**.

### Docker build reliability

- **Geo data downloads:** для `geoip.dat`, `geosite.dat`, `geoip_IR.dat`, `geosite_IR.dat`, `geoip_RU.dat`, `geosite_RU.dat` добавлены до **3 попыток** загрузки.
- **Явный fail в CI:** при неуспешной загрузке после 3 попыток сборка теперь завершается ошибкой, чтобы проблема была видна в job/log.

2. Подтяните образы **`1.2.3`** / **`latest`** и пересоздайте контейнеры панели/ноды.

```yaml

  sharx:
    image: {{SHARX_TAG_LATEST}}
  sharxnode:
    image: {{SHARXNODE_TAG_LATEST}}
  postgres:
    image: {{HARBOR_HOST}}/{{HARBOR_PROJECT}}/postgres:latest
```
