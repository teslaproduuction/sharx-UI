## Release v1.3.1

**Release Date:** May 6, 2026

Релиз относительно **v1.3.0**: настраиваемые **заглушки в текстовой подписке** (кастомные строки для отключённого клиента, истечения срока, лимита трафика, пустых хостов, лимита HWID), **JSON-подписка** с уведомлением при блокировке по HWID (blackhole), **HWID** уникален только в паре с клиентом панели (одно устройство может быть в нескольких учётках), редактор в **конструкторе подписки**, **`gofmt`** по коду. Новых миграций БД нет.

### New Features

- **Custom remarks (подписка)**
  - В конфиге v2 страницы подписки: **`showCustomRemarks`** и **`customRemarks`** (списки строк по сценариям: истёк срок, лимит трафика, отключён пользователь, нет хостов, превышен лимит HWID; поле **HWIDNotSupported** в конфиге и UI — заготовка).
  - Текстовая подписка: вместо реальных узлов — **фиктивные VLESS-строки** (`vless://…#текст`), клиенты показывают fragment как заголовок узла.
  - JSON-подписка: при ошибке по HWID и непустых строках — минимальный конфиг с **blackhole** outbounds, теги = текстам заметок.

- **Панель**
  - **`CustomRemarksEditor`** в конструкторе подписки (`SubscriptionBuilder`), схема в **`sharxSubpageConfig.ts`**.

- **HWID**
  - Снята глобальная проверка «HWID уже у другого клиента»; учёт по-прежнему и лимиты **`MaxHWID`** — **внутри одного клиента**.

### Improvements

- Отключённый по `sub_id` клиент: при настроенных строках для **DisabledUsers** — ответ placeholder’ами и трафик из записи клиента.

### 📦 Upgrade from v1.3.0

1. Сделайте **бэкап** базы данных и `.env`.
2. Поднимите панель с версией **`1.3.1`** (образы **`1.3.1`** / **`latest`**) — миграций **нет**.
3. После старта проверьте:
   - **Конструктор подписки** — custom remarks и переключатель показа;
   - **Подписка** — сценарии disabled / expired / limited / empty hosts / HWID limit (plain и при необходимости JSON);
   - при необходимости — **один HWID на нескольких клиентах**.

```yaml
services:
  sharx:
    image: {{SHARX_TAG_LATEST}}
  sharxnode:
    image: {{SHARXNODE_TAG_LATEST}}
  postgres:
    image: {{HARBOR_HOST}}/{{HARBOR_PROJECT}}/postgres:latest
```
