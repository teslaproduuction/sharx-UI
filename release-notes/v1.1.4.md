## Release v1.1.4

**Release Date:** April 27, 2026

Патч-релиз относительно **v1.1.3**: исправление регистрации нод в PostgreSQL.

### Исправления

- **База данных / ноды:** колонка `nodes.enable` приводится к типу **BOOLEAN** (миграция `0033`). Ранее в миграции `0028` она была **INTEGER**, из‑за чего при добавлении ноды драйвер не мог записать значение `true` в `int4` (`unable to encode true into binary format for int4`). После обновления панели миграция применится при старте; при необходимости тот же `ALTER` можно выполнить вручную на БД.

1. Сделайте **бэкап** PostgreSQL и `.env`.
2. Подтяните образы **`1.1.4`** / **`latest`** и пересоздайте контейнер панели (миграции выполняются при запуске).

