## Release v1.1.6

**Release Date:** April 27, 2026

Патч-релиз относительно **v1.1.5**: подписка и share-ссылки (VLESS flow, XHTTP padding), правки панели и WireGuard.

### Исправления и поведение

- **Subscription / VLESS `flow`:** в ссылках подписки снова попадает `flow` клиента; если в записи клиента пусто — берётся из JSON inbound (`settings.clients[].flow`). Закрывает [SharX#71 — В Subscription нет Flow](https://github.com/konstpic/SharX/issues/71).
- **Share links / XHTTP:** для VMess безопасное чтение `mode` и перенос настроек xhttp padding в JSON ссылки; для VLESS / Trojan / SS — параметры `x_padding_bytes` и `extra` (как в 3x-ui) для клиентов Xray/sing-box.
- **Панель:** доработки страниц клиентов и inbounds, географии нод, схем Monaco для Xray, метаданные приложения в шапке.
- **WireGuard:** расширения panel/service (в т.ч. работа с пирами клиентов), обновления переводов en/ru.
